# Dedalus Marketplace Info

## Short Description
Detect licenses and check compatibility against policies

## Long Description
Detect SPDX licenses from package metadata or file contents, check compatibility against allow/deny policies, and generate NOTICE files for attribution. Tools: detect_licenses, check_compatibility, generate_notice.
